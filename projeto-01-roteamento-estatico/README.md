# Projeto 01 - Comunicação entre duas Sub-Redes com Roteamento Estático 

Esse projeto tem como objetivo a criação de um laboratório utilizando o Cisco Packet Tracer para simular a comunicação entre duas Sub-Redes.

Para criar a topologia desta rede, foram utilizados três computadores, dois servidores, dois switches e dois roteadores. 

# Objetivos 

- Configurar interfaces de roteadores Cisco.
- Aplicar conceitos de sub-redes em uma rede IPv4.
- Estabelecer rotas estáticas para comunicação entre sub-redes.
- Testar a conexão usando ferramentas de diagnóstico.
