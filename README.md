# Laboratórios de Redes de Computadores

Bem-vindo ao meu repositório de estudos em Redes de Computadores.

Aqui reúno os laboratórios que desenvolvo no Cisco Packet Tracer para praticar conceitos de infraestrutura, roteamento, switching e troubleshooting. O objetivo é registrar minha evolução técnica e documentar cada projeto de forma organizada.

## Projetos

- Projeto 01 | Comunicação entre duas sub-redes com roteamento estático | ✅ Concluído 
- Projeto 02 | DHCP | ⏳ Em desenvolvimento |
- Projeto 03 | OSPF | ⏳ Planejado |
- Projeto 04 | VLANs | ⏳ Planejado |
- Projeto 05 | ACLs | ⏳ Planejado |

## Tecnologias e conceitos

- Cisco Packet Tracer
- IPv4 e Subnetting
- Roteamento Estático
- OSPF
- VLANs
- DHCP
- ACLs
- NAT/PAT
- Comandos de diagnóstico (`ping`, `tracert`, `show`)

## Objetivo

Construir um portfólio prático de Redes de Computadores, documentando configurações, testes e aprendizados obtidos em cada laboratório.
